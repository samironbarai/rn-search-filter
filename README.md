## React native search filter on flatlist

## Clone this repo
```
git clone https://github.com/samironbarai/rn-search-filter.git
```

## Install dependency
```
$ cd rn-search-filter
$ npm install
```

## Put API endpoint
```
$ replace yourwebsite.com with your website
```

## Run the project
```
react-native run-android
```
